## Usage

 Run the CLI using

```sh
npx new-node-cli

# OR for global install
new-node-cli
```

_Then answer the following questions to generate a Node.js CLI._

```sh
CLI name?
CLI command?
CLI description?
CLI version?
CLI repository?
CLI license?
CLI author name?
CLI author email?
CLI author URL?
````
